# Postcard-portrait-plotter

Run as:

```bash
zx index.mjs
```

Options:

- `-f <file>`
- `-p` to plot it

## Pyhton Environment

[Creating a virtual env](https://packaging.python.org/en/latest/guides/installing-using-pip-and-virtual-environments/#creating-a-virtual-environment)

```bash
python3 -m venv env
```
